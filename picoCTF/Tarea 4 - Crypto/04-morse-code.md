
## Descripción

Morse code is well known. Can you decrypt this? Download the file [here](https://artifacts.picoctf.net/c/79/morse_chal.wav). Wrap your answer with picoCTF{}, put underscores in place of pauses, and use all lowercase.

## Solución

Inicio descargando el archivo de audio usando el comando wget.

De acuerdo a la pista uso Audacity para abrirlo y ver las líneas.
Uso un decodificador de código morse para decodificarlo.
Obtengo la bandera solamente lo dejo en minúsculas, agrego los _  en los espacios y el formato de la bandera:
`picoCTF{wh47_h47h_90d_w20u9h7}`
