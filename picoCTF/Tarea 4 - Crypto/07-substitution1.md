
## Descripción

A second message has come in the mail, and it seems almost identical to the first one. Maybe the same thing will work again. Download the message [here](https://artifacts.picoctf.net/c/181/message.txt).

## Solución

Inicio descargando el mensaje usando el comando wget.

Copeo el mensaje cifrado y voy a la página dcode, donde utilizo mono alphabetic substitution en la sección automática.
Arreglo ahí un caracter que salió diferente, obtengo la bandera:

`picoCTF{FR3QU3NCY_4774CK5_4R3_C001_4871E6FB}`
